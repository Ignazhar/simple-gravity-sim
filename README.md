# simple-gravity-sim
Simple gravity simulation using C++ and SFML library.

### Windows Powershell execution:

Enter the src directory:

```PS ...> cd src```

Compile:

```PS ...> mingw32-make```

Execute:

```PS ...> ./main.exe```
